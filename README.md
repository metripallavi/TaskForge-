# TaskForge

TaskForge is a cloud-native job processing platform inspired by:

- Celery
- Sidekiq
- Temporal
- AWS Batch

The project demonstrates:

- Backend Engineering
- DevOps Engineering
- Cloud Engineering
- Platform Engineering
- Distributed Systems

## Tech Stack

### Backend

- Python
- FastAPI
- SQLAlchemy
- Alembic
- Pydantic

### Database

- PostgreSQL

### Queue

- Redis

### Background Processing

- Celery

### Infrastructure

- Docker
- Kubernetes
- Helm
- Terraform

### Cloud

- Azure
- AKS

### Observability

- Prometheus
- Grafana

### CI/CD

- GitHub Actions
